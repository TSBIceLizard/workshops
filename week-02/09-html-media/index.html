Media

Ethicality Around sourcing third party images:

Make sure that if you source an image or piece of content from elsewhere that you check the license and terms of using the piece of media. Typically, this will involve crediting the author but there are occasions (licensed music for example) where this is not enough.

I will list some examples of free libraries where media can be sourced below.

  • Pixabay contains a multitude of different media types.
  • BBC Sounds hosts the BBC's vast historical collection of sounds that were recorded over the years by their sound team.
  • Unsplash contains royalty free stock images.

It's good practice to always look at the terms and conditions or licensing details about wherever you're sourcing content from!

Video Element Attributes:

  • poster="" - Sets the video thumbnail.
  • muted - Auto mutes the sound.
  • autoplay - Plays when the page loads.
  • loop - Loops the content automatically when the video reaches the end.

Audio Element Attributes:

  • controls - Renders a way to interact with the sound controls. Including Play, Progress, Mute and Change volume.
  • muted - Mutes the sound by default.
  • autoplay - Triggers the playing of the sound to start when the webpage loads.
  • loop - Repeats the sound when the sound has finished playing.

Image Element Attributes:

  • alt="" - A description of the image.
  • width="number" - Controls the size in pixels of the width of the image. It's favourable to do this in CSS instead of HTML!
  • height="number" - Same as above but affects height! Better to use CSS still.
  • usemap="#myimagemap" - An image map can be used to select specific regions of an image. Very useful if you want certain areas to be clickable. for more info!
